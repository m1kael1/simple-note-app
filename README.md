
# Simple-Notes-App

Ini adalah aplikasi catatan sederhana berbasis web dengan ReactJs.
## Demo 
 [Simple-Notes-App](https://simple-note-app-theta.vercel.app)
## Features

- Create Note
- Edit Note
- Search Notes

## Installation

### Clone Project

```bash
 git clone https://github.com/m1kael1/simple-note-app.git
```
```bash
 cd simple-note-app
```
### Install Dependensi
```bash
 npm install
```
### Run development
```bash
 npm run dev
```
